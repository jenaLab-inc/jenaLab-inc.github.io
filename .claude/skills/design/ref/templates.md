# Screen Type & Project Templates (Parts 9-10)

> Reference file for `design/SKILL.md` — See main skill for core philosophy and AX system.

## Part 9: Screen Type Templates

### Home / Landing
- Hero: one landscape-like image or illustration
- Info area: very ordered
- CTA: clear, not exaggerated
- Top: structure over decoration; decoration density increases lower

### Browse / List
- Background: as simple as possible
- Card/row spacing: generous
- Thumbnails: landscape or material-quality focused
- Filter: lightweight
- Selected state: quiet natural color

### Detail Screen
- Top image/scene: large
- Body text: generous whitespace
- Fixed action area: visually light even when sticky
- Decoration: limited to near-title or section start only

### Input / Reservation / Payment
- Strip decoration almost entirely
- Warm background tone only
- Label / validation / button clarity = top priority
- Section dividers: thin lines + generous spacing
- Restoration of atmosphere elements allowed only on success/completion screen

### Settings / Profile / My Page
- Japanese-style order dominant
- Tone: paper + moss/slate
- Icons: thin, quiet
- Watercolor/vintage: extremely restrained
