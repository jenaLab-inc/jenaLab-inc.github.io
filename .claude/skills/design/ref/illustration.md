# Illustration and Image Guide (Part 7)

> Reference file for `design/SKILL.md` — See main skill for core philosophy and AX system.

## 7.1 Urban Sketch / Watercolor Rules

```
Lines first, then color
Draw only the essential outline — leave surrounding space open
Watercolor creates mass with soft edges
Near center: detailed; periphery: open and airy

Use in:
  - Empty state
  - Onboarding
  - Hero illustration
  - Section divider images
  - Background pattern zones
  - Card thumbnail frames

NEVER use in:
  - Behind primary CTAs
  - Form/input areas requiring text legibility
  - High-density data screens
  - Primary interaction zones of productivity features
```

## 7.1.1 Human Scribble — Anti-AI Authenticity (2026)

An evolution of Urban Sketch that deliberately signals **"a human made this"** — scribbles on logos, doodles over photos, sketch-like overlays. In an AI-saturated landscape, imperfection becomes a trust signal.

```
Maps to: AX-SK >= 2

Allowed scribble techniques:
  - Hand-drawn circle/underline emphasis on key words (SVG path, irregular stroke)
  - Doodle overlay on product/hero photos — sketch lines sitting on top of photography
  - Scribbled annotation marks (arrows, stars, checkmarks) as UI accents
  - Logo variant with hand-drawn modification for campaigns or seasonal pages
  - Sketch-style section dividers replacing geometric rules

Rules:
  - Scribble elements must feel SPONTANEOUS, not perfectly imperfect — avoid overly uniform "fake hand-drawn"
  - Use as ACCENT (5-10% of visual weight) — not as the dominant visual language
  - Color: ink-charcoal, sepia-wash, or soft-brass — never bright or neon
  - Stroke width: 1-2px for overlays, 2-3px for standalone marks
  - DO NOT apply to functional UI elements (buttons, inputs, nav) — scribble is editorial, not interactive
  - DO NOT combine with Light Skeuomorphism on the same element — choose one texture approach
  - AX-JP >= 4: limit scribble to one zone per page (hero or divider, not both)
```

## 7.2 Landscape / Scene Image Rules

```
Preferred qualities:
  - A place you can linger in — not a grand panorama
  - Wind, trees, grass, windows, terraces, rain, interior lighting
  - Space and air quality over people
  - Quiet-hour light
  - "A scene where a cup of coffee belongs"

Interior images:
  - Warm bulb light, indirect lighting, wood, wall reflections, fabric texture
  - Few props, comfortable sense of place, uncluttered
```

## 7.3 Decorative Motif Rules

```
Forms: thin flowing plant stems, flower buds, arches, frame lines
Placement: concentrate in one zone only (top / bottom / corner)
Limit: one decorative motif type per screen
Style: silhouette / line-based — not filled patterns
Empty space: linear decoration that preserves negative space > dense fill patterns
```

## 7.4 Baroque Translation Rule

Baroque != ornate decoration copied directly. In UI, translate Baroque as:
- **Depth** through layered surfaces
- **Composition** with a clear central axis
- **Rhythm** in layout flow
- **Material quality** through texture and color
- **Lighting drama** through shadow and illumination contrast
