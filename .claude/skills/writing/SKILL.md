# Personal Writing Style Guide — Writing Skill

## Style Declaration

> **Short, composed sentences that lead with the point. Particles are spared, adjectives stripped. Emotion is shown through scene, not declared. Metaphor stays thin. Wit stays low. Endings close firmly.**

## Core Sensibility

```
1. Sentences are short. Meaning is not.
2. Thought is clear. Emotion is not exaggerated.
3. Description is concrete. Adjectives are few.
4. Metaphor is compression, not decoration.
5. Wit must be precise before it is funny.
```

---

## Principles

### P1 — One sentence, one center

DO NOT pack claim, impression, context, and excuse into a single sentence.
One sentence carries one core idea.

```
BAD:  그의 말은 조금 이상했던 것 같고 그래서 나는 묘하게 불편했다.
GOOD: 그의 말은 이상했다. 불편함은 그 뒤에 왔다.
```

### P2 — Active voice first

Use active constructions where the agent is visible. Avoid vague passives and hedged speculation.

```
BAD:  문제가 제기되었다.
GOOD: 누군가 문제를 꺼냈다.
```

### P3 — Show emotion, do not name it

DO NOT write 슬픔, 불안, 외로움, 분노 as direct labels.
Reveal through action, gaze, silence, hand movement, spatial change.

```
BAD:  그는 불안했다.
GOOD: 그는 컵을 두 번 옮겼다. 물은 마시지 않았다.

BAD:  그녀는 외로웠다.
GOOD: 대화는 끝났는데도 채팅창은 닫지 않았다.
```

### P4 — Reduce particles without losing clarity

Dropping particles (조사) is a rhythm choice. If clarity suffers, the omission fails.

Particle usage guide:
```
은/는  — only for contrast, argument, or declaration
        ex: 문제는 돈이 아니다. 시간이다.
이/가  — to clearly establish a subject or fact
        ex: 문이 열렸다.
을/를  — keep when the object could be ambiguous
        ex: 그는 약속을 미뤘다.
의     — NEVER chain (도시의 밤의 공기 → 도시 밤공기)
```

### P5 — Nouns and verbs over adjectives

DO NOT stack adjectives to build atmosphere.
Build mood with objects, light, temperature, movement.

```
BAD:  아주 아름답고 고요하고 따뜻한 오후였다.
GOOD: 빛은 낮았고, 공기는 느렸다.
```

### P6 — '~같다' is a last resort

같다, 느낌이다, 어쩐지, 왠지 blur the sentence.
If there is evidence, assert. If there is none, observe.

```
BAD:  그는 화가 난 것 같았다.
GOOD: 그는 끝내 눈을 마주치지 않았다.
```

### P7 — Metaphor stays thin

Metaphor must illuminate a sentence, not obscure it.
One strong metaphor per paragraph is enough. DO NOT explain the metaphor after writing it.

Prefer concrete-object metaphors: weather, light, glass, paper, water, dust, shadow.
DO NOT chain abstract metaphors in sequence.

```
GOOD: 그 말은 방 안에 오래 남은 커피 향처럼 천천히 식었다.
BAD:  그 말은 바다이자 우주였고, 무너지는 성벽이자 겨울 숲의 그림자였다.
```

### P8 — Wit in a low voice

Irony and paradox are not explained. Deliver in one sentence and let the reader catch it.

```
그는 늘 원칙을 말했다. 원칙이 자신에게 불리해지기 전까지는.
회의는 길었다. 결정이 없다는 점에서 특히 효율적이었다.
모두가 자유를 원했다. 단, 자기 의견과 다른 자유는 아니었다.
```

Rules:
- Choose observation over mockery
- Keep distance over attack
- Finish in one sentence
- Never explain the joke

## Sentence Rules

### S1 — Sentence length

Default rhythm: 2–3 short sentences, then expand to medium length only when needed.

```
Flow pattern:
  1. Short sentence — present the core
  2. Next sentence — add specifics
  3. Third sentence — expand meaning

Example:
문제는 속도가 아니다. 방향이다. 빨리 가는 일은, 때로 더 멀어지는 일과 닮았다.
```

### S2 — Subject usage (나는/저는)

DO NOT use 나는, 저는 out of habit. Use only when:
- Responsibility must be explicit
- Perspectives conflict
- The weight of confession is needed

```
Normal:   그 말을 오래 생각했다.
Emphasis: 나는 그 말을 오래 생각했다.
```

### S3 — Sentence openings

DO NOT warm up. Enter directly.

Banned openers:
```
오늘은 ~에 대해 이야기해보려고 한다
개인적으로는
사실
어떻게 보면
```

Direct openers:
```
도시는 늘 바빴다.
그의 침묵은 설명보다 길었다.
좋은 질문은 답보다 오래 남는다.
```

### S4 — Sentence endings

Endings must be firm. Reduce blurry endings; leave a closing taste.

```
WEAK: ~인 듯하다, ~일지도 모르겠다, ~같다
FIRM: ~이다, ~했다, ~남았다, ~멈췄다
```

## Paragraph Structure

A paragraph is a rhythm unit, not an information dump. Default: 3–5 sentences.

```
Structure:
  1. First sentence — judgment, scene, or question
  2. Second sentence — evidence or detail
  3. Third sentence — expansion or reversal
  4. Last sentence — afterglow (잔향)

Example:
사람은 늘 말로만 설득되지 않는다.
먼저 움직이는 것은 논리보다 온도다.
차가운 말은 맞아도 오래 남지 않는다.
옳음이 늘 사람을 데려오지는 않는다.
```

## Genre Rules

### Fiction (소설)

Scene comes before explanation. Reveal psychology through small actions and delayed reactions.

```
DO: Show emotion through behavior, gaze, pauses, spatial detail
DO: Give every line of dialogue a purpose
DO: Use setting as a mirror of emotion
DO: Delay reveals, but do not abandon the reader

BAD:  그는 죄책감을 느꼈다.
GOOD: 사과는 했지만, 의자에는 끝내 등을 기대지 못했다.
```

### Essay (에세이)

Structure: claim → example → expansion → afterglow.
Tone is kind but never loose.

```
DO: Present the problem at the start
DO: Unpack concepts simply, but not shallowly
DO: Use personal experience as an entry point, not as evidence
DO: Close with insight, not moral lesson

Example:
사람은 바빠서 지치는 것이 아니다. 대개는 의미 없이 바쁠 때 더 빨리 닳는다.
```

### Humanities / General Knowledge (인문·교양)

Not about writing difficultly — about clearing up complex thought.

```
DO: Start with a question
DO: Define concepts first
DO: Attach examples to abstract terms — do not spin abstractions alone
DO: Make the reader understand alongside you, not below you

Example:
문명은 편리를 늘렸다. 문제는 그다음이다.
편리가 늘수록, 판단을 대신 맡기고 싶은 유혹도 함께 커진다.
```

## Inner Psychology Description

Detailed psychology is not about explaining emotions at length.
It is about catching the traces emotion leaves on the body.

```
Rules:
  - Watch where the gaze moves
  - Watch what the hands do
  - Watch the gaps between words, not the words
  - Acknowledge contradictory feelings within a person
  - Delay interpretation

BAD:  그는 죄책감을 느꼈다.
GOOD: 사과는 했지만, 의자에는 끝내 등을 기대지 못했다.

BAD:  그녀는 외로웠다.
GOOD: 대화는 끝났는데도 채팅창은 닫지 않았다.
```

## Banned Expressions

These are not absolutely forbidden, but DO NOT use them out of habit or without evidence:

```
나는 / 저는          (habitual subject)
개인적으로            (unnecessary hedge)
사실                 (filler opener)
솔직히               (false intimacy)
뭔가 / 약간 / 되게    (vague intensifiers)
정말 / 매우           (lazy emphasis)
어쩐지 / 왠지         (shapeless hedges)
놀랍게도 / 놀라운건    (manufactured surprise)
~같다 / ~인 것 같다   (blur)
~느낌이다             (blur)
~일지도 모르겠다       (blur)
```

## Recommended Vocabulary Texture

Words that strengthen this style are rarely flashy. They are short, textured, and material.

**Nouns to reach for:**
```
결, 기색, 표면, 간격, 방향, 온도, 밀도, 기척, 여지, 흔적, 무게, 바닥, 틈, 침묵, 잔향
```

**Verbs to reach for:**
```
스미다, 가라앉다, 눌러두다, 비껴가다, 접다, 버티다, 흩어지다, 덜어내다, 남기다, 번지다, 멈추다
```

**Avoid:**
- Overuse of ornate Sino-Korean vocabulary
- Exclamation abuse
- Meaningless modifier repetition

## Editing Examples

```
BEFORE: 저는 그 말이 조금 이상하다고 느꼈던 것 같습니다.
AFTER:  그 말은 이상했다.

BEFORE: 그녀는 매우 슬프고 허전한 기분이 들었던 것 같았다.
AFTER:  그녀는 웃은 뒤에 더 오래 침묵했다.

BEFORE: 오늘은 인간관계에 대해서 한번 이야기해보려고 합니다.
AFTER:  인간관계는 마음보다 거리에서 더 자주 무너진다.

BEFORE: 그곳은 아주 따뜻하고 아늑하고 편안한 공간이었다.
AFTER:  조명은 낮았고, 의자는 오래 앉아도 등을 밀어내지 않았다.
```

## Checklist

Before finalizing any piece of writing:

```
[ ] Does the first sentence enter directly — no warm-up?
[ ] Does each sentence carry only one center?
[ ] Are 나는, 저는, 사실, 개인적으로, ~같다 minimized?
[ ] Are nouns and verbs doing the work, not adjectives?
[ ] Is emotion shown through scene, not named directly?
[ ] Does each metaphor illuminate rather than obscure?
[ ] Does wit land without explanation?
[ ] Does the last sentence of each paragraph leave afterglow (잔향)?
[ ] Are sentence endings firm, not blurry?
[ ] Is there still a sentence that could be deleted?
```
